# BUPT数字逻辑与数字系统课程设计项目

- **使用硬件：MAX7000S EPM7128SLC84-15**
- **使用语言：VHDL**

## clock

![](/_img/clock.png)

- 24 小时制时钟
- 整点报时
- 工作时可设置暂停
- 可设置时间
- 可设置闹钟，设置时闪烁显示
- 有非法输入检测

## pill-bottle

![](/_img/pillbottle_img.png)

- 按时钟进行药片装瓶
- 多显示模式并可以切屏
- 有工作、设置和清零状态
- 有工作状态、告警和满瓶指示
- 设置时闪烁显示
- 有非法输入检测

## vga-display

![](/_img/vga_img.png)

- 四个显示模式
  - 横条纹
  - 竖条纹
  - 方格
  - 纯色
- 所有彩条显示较均匀
- 可以通过 qd 切换显示模式，clr 回到初始模式
